# Hydraulics Formula Handbook (Web)

Academic SI reference site — 24 chapters, KaTeX equations, worked examples, calculators, selective R3F schematics.

## Production

https://hydraulics-handbook-web.vercel.app

## Local

```bash
npm install
npm run dev
```

## Stack

Next.js App Router · React · Tailwind · KaTeX · GSAP · AutoAnimate · React Spring · React Three Fiber
