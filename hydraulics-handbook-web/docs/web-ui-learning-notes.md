# Ghi chú — Tư duy làm Web / UI / Animation

Tài nguyên tham khảo để học và làm web hiện đại: tư duy UI, 3D/WebGL, scroll storytelling, animation.

---

## Tư duy làm Web & UI

- **taste-skill** — Skill / tư duy thẩm mỹ UI  
  https://github.com/Leonxlnx/taste-skill.git

- **Aceternity UI Templates** — Template tham khảo  
  https://ui.aceternity.com/templates

- **React** — Nền tảng thư viện UI  
  https://github.com/react/react.git

---

## 3D / WebGL

- **React Three Fiber** — React renderer cho Three.js  
  https://github.com/pmndrs/react-three-fiber.git

---

## Scroll storytelling

- **GSAP** — Animation / scroll storytelling  
  https://github.com/greensock/GSAP.git

---

## Animate chuyển động

- **AutoAnimate** — Animate tự động khi DOM thay đổi  
  https://github.com/formkit/auto-animate.git

- **Lottie Web** — Render animation từ After Effects (JSON) trên web  
  https://github.com/airbnb/lottie-web.git

---

## Hiệu ứng di chuyển component (user-driven)

- **React Spring** — Physics-based animation, kéo/thả, gesture  
  https://github.com/pmndrs/react-spring.git

---

## Gợi ý lộ trình học

1. Nắm React cơ bản
2. Xem taste-skill + Aceternity để định hướng thẩm mỹ / layout
3. AutoAnimate cho motion nhẹ khi list/form đổi
4. Lottie Web khi cần animation thiết kế sẵn (icon, illustration, micro-interaction)
5. React Spring khi cần tương tác kéo, spring, gesture
6. GSAP khi cần scroll storytelling, timeline phức tạp
7. React Three Fiber khi cần 3D / WebGL
